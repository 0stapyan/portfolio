# Project: Amateur Football League Management System

- This application is designed to automate the core operations of an amateur football league. It enables team creation, match scheduling, referee assignment, match reporting, and automatic league table generation.

## Key Features:

- 📋 Team and player creation
- 📅 Match fixture generation by age categories
- 🧑‍⚖️ Automatic referee assignment based on experience
- 📝 Match report submission with statistics update
- 🏆 League table generation with points, goal difference, and results

**Structure**:

- models/ — classes for teams, players, matches, and referees
- logic/ — core business logic (schedule, table, referee assignment)
- tests/ — basic unit tests to verify core module functionality
- main.cpp — a demonstration of the full application flow

### The project is written using object-oriented principles and clean modular design.
